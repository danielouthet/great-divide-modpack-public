## üîß Installing Forge (Minecraft 1.20.1)

2. Select **Minecraft 1.20.1**
3. Download the **Installer**
4. Open the installer
5. Select **Install Client**
6. Click **OK**
7. Wait for installation to finish

‚ö†Ô∏è Do NOT download ‚ÄúUniversal‚Äù or ‚ÄúChangelog‚Äù files.

## üß† Allocate More RAM (Recommended)

1. In **Minecraft Launcher**, go to **Installations**
2. Click **Edit** on your Forge installation
3. Click **More Options**
4. Find:-Xmx2G
5. 5. Change it to: -Xmx6G
   6. - Use **4G** for lower-end systems
- You can go higher if you want but Do NOT exceed half your total system RAM
6. Click **Save**

4. Copy new `.jar` files into the `mods` folder
5. Launch Minecraft

‚ö†Ô∏è Never mix old and new mods.
